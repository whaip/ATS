# 项目信息

## 项目概述

本项目为“国家管网西部管道控制卡件（模块）通用测试系统”，文档版本为 `V1.0`，时间为 `2024`。系统面向控制板卡、功能模块及相关电子单元的通用测试场景，目标是在统一平台上完成板卡识别、任务配置、自动接线、信号激励、数据采集、故障诊断、温度监测、结果记录与对外数据发布。

## 建设目标

- 建立一套面向多类控制卡件和电子模块的通用测试平台。
- 统一板卡图像识别、元件检测、故障诊断和设备联动流程。
- 支持 PXI 仪器、相机、红外测温、任务日志和统计接口的集成部署。
- 为后续新增板卡、测试策略和诊断算法预留扩展能力。

## 主要功能

结合技术方案文档，当前系统重点覆盖以下能力：

- 板卡图像采集、板卡识别与模板库管理。
- 元器件检测、位号识别和任务配置。
- 测试策略配置与测试流程编排。
- 激励输出、采样采集、多设备协同控制。
- 故障模式识别、诊断结果生成与测试报告输出。
- 红外温度监测、预警与停机阈值联动。
- 测试日志归档、统计分析以及 HTTP/TCP/UDP 数据发布。

## 系统组成

根据方案文档，系统整体由上位机软件、PXI 测试平台、激励与采集设备、视觉采集设备、温度监测设备以及配套夹具构成。典型硬件组成包括：

- 工控机或上位机软件平台。
- PXI 机箱及相关板卡。
- `JY5711` 输出设备。
- `JY5322 / JY5323` 采样设备。
- `JY8902` 测量设备。
- 高清相机与红外相机。
- 固定夹具、连接治具与测试工装。

## 项目价值

该系统把“识别、配置、测试、诊断、记录、发布”打通到一条统一流程内，减少人工切换工具和人工判读带来的效率损耗，同时为后续板卡类型扩展、策略插件扩展和诊断算法迭代提供稳定的软件基础。
